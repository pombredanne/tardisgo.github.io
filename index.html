TARDIS Go transpiler

TARDIS Go is a Go->Haxe transpiler, but aspires to eventually target other languages too.

The objective of the project is to allow Go code to be re-used in other contexts, initially alongside code written in Haxe, JavaScript, ActionScript, C++, Java, C# or PHP (which are all generated by the Haxe compiler).

On the client-side, using Haxe libraries like OpenFL allows TARDIS Go to be deployed on a very wide range of platforms, including iOS, Android, BlackBerry, Tizen, WebOS, Windows, Mac, Linux, Flash and HTML5.
