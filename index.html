TARDIS Go transpiler

TARDIS Go is a Go->Haxe transpiler. The Haxe compiler then generates equivalent code for JavaScript, ActionScript, C++, Java, C#, PHP and Neko.

The objective of the project is to save time and energy by enabling transpilations of the same TARDIS Go code to form part of an application written in any of these target languages.

On the client-side, using Haxe libraries like OpenFL allows TARDIS Go to be deployed on a very wide range of platforms, including iOS, Android, Tizen, Windows, Mac, Linux, HTML5 (example at the top of this page) and Flash.
