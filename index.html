TARDIS Go compiler holding page

This project is called TARDIS Go because of this blog entry. It is planned to be published by the time of the "Write your own Go compiler" talk.

For more about the project see the slides here. Sample code is available here. Target code is generated by a sequence of translations Go -> SSA -> HAXE (maybe plus an OpenFL haxe wrapper) -> JS/Flash/C++/Java/C#/PHP/Neko.
