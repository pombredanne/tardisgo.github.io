TARDIS Go transpiler

TARDIS Go is currently a Go->Haxe transpiler, but aspires to eventually target other languages too. It is called "TARDIS" Go because of this blog entry.

The project should be published by 23rd January 2014, when there will be a preview of the FOSDEM'14 "Write your own Go compiler" talk at the Go London User Group meeting.

Target code is generated by a sequence of translations Go -> SSA -> HAXE (maybe plus an OpenFL haxe wrapper) -> JS/Flash/C++/Java/C#/PHP/Neko. For more about the project context see the slides here.
