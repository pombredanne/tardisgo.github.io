TARDIS Go compiler holding page

This Go->Haxe transpiler project is planned to be published by 23rd January 2014, when there will be a preview of the FOSDEM'14 "Write your own Go compiler" talk at the Go London User Group meeting.

It is called "TARDIS" Go because of this blog entry.

Target code is generated by a sequence of translations Go -> SSA -> HAXE (maybe plus an OpenFL haxe wrapper) -> JS/Flash/C++/Java/C#/PHP/Neko. For more about the project context see the slides here.

Teaser example below, showing a Go goroutine reacting to a channel of events generated by your mouse. This version is in JS, a Flash version is available here. Both were created from exactly the same source code, which is available here in subdirectory haxe-openfl/gohandlingmouseevents.
