TARDIS Go transpiler

TARDIS Go is currently a Go->Haxe transpiler, but aspires to eventually target other languages too. The objective of the project is to allow Go code to be re-used in other contexts, initially alongside code written in Haxe, JavaScript, ActionScript, C++, Java, C# or PHP. On the client-side, using the Haxe library OpenFL allows TARDIS Go to be deployed on a very wide range of platforms, including iOS, Android, BlackBerry, Tizen, WebOS, Windows, Mac, Linux, Flash, HTML5 and Emscripten.

The project should be published by 23rd January 2014, when there will be a preview of the FOSDEM'14 "Write your own Go compiler" talk at the Go London User Group meeting. It is called "TARDIS" Go because of this blog entry. A set of slides about the project context are here.
