TARDIS Go transpiler

TARDIS Go is a Go->Haxe transpiler, but aspires to eventually target other languages too. The objective of the project is to allow Go code to be re-used in other contexts, initially alongside code written in Haxe, JavaScript, ActionScript, C++, Java, C# or PHP. On the client-side, using the related Haxe libraries OpenFL and Lime allows TARDIS Go to be deployed on a very wide range of platforms, including iOS, Android, BlackBerry, Tizen, WebOS, Windows, Mac, Linux, Flash, HTML5 and Emscripten.

The author will deliver a talk called "Write your own Go compiler" at FOSDEM'14 on February 2nd, with a preview of that talk at the Go London User Group meeting on January 23rd.
